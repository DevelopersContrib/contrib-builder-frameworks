Join, Contribute and Receive Equity

Contrib is a new way to contribute for equity points, and in this instance, code. There is a better way for your talents and skills to be utilized and that is to be able to have equity points or ownership of a url.

In Contrib, we are deeply integrated into VNOC, which houses 20k of premium,underutilized and unused urls or domains. We need people to contribute and create value to these great domains and what better way to contribute, is through code.

How To Get Started


Join our Developers.Contrib.com community of developers and get an api key

Fork any of our frameworks from Docker or Github

Start creating your own for any of the domains we have on our portfolio

Submit your framework for approval. All submitted frameworks are manually checked

Get equity points for every code contribution
